# GitHub User Finder

**Автор:** Карпенко Евгения

## Описание

GitHub User Finder — это графическое приложение на Python для поиска пользователей GitHub с использованием официального GitHub API. Приложение позволяет искать пользователей, просматривать результаты поиска, добавлять пользователей в избранное и сохранять их локально в JSON-файл.

## Возможности

- 🔍 Поиск пользователей GitHub по логину
- 📋 Отображение результатов поиска в таблице
- ⭐ Добавление/удаление пользователей в избранное
- 💾 Автоматическое сохранение избранного в JSON
- 📊 Детальная информация о пользователе из избранного
- ✅ Валидация ввода (поле поиска не может быть пустым)
- 🎨 Удобный интерфейс с вкладками
